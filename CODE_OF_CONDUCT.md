# Code Of Conduct

This project expects direct, respectful collaboration.

## Expected Behavior

- Be clear and constructive.
- Focus criticism on code, design, documentation, and process.
- Assume mistakes can be fixed.
- Respect different experience levels and operating environments.
- Keep issue and pull request discussions on topic.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without consent.
- Repeatedly derailing technical discussions.
- Sharing secrets, credentials, or private infrastructure details.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
or restrict participation when behavior harms the project or its contributors.

Report conduct concerns to the repository maintainers through a private channel
when one is available.
