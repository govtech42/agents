# Changelog

All notable changes to this project are documented here.

## v1.0.0 - 2026-06-29

- Introduced `aai`, a Docker Compose based CLI for installing personal AI agent
  services.
- Added documented recipes for OpenClaw, Hermes Agent, and Paperclip.
- Added GBrain as an optional HTTP MCP memory layer for supported agents.
- Added local and remote SSH install flows, including saved targets and dry-run
  previews.
- Added generated project layout for `data/`, `config/`, `templates/`,
  `docker-compose.yml`, and `aai.json`.
- Added `.env.example` and gitignore coverage for local secrets.

## Unreleased

- Refined public documentation and repository metadata for the documented agent
  set.
- Added basic project documentation for architecture, contribution, security,
  support, and conduct.
